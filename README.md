To get started 
```
git clone https://github.com/Ruban2205/Get_weather_Web_app.git
```

To install the dependencies: 
```
npm install 
```

To run the development server: 
```
npm run dev
```

To Build the app run this command: 
```
npm run build
```

Example of `.env` file 
```
VITE_KEY='YOUR_API_KEY'
```
